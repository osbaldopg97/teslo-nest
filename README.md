# Teslo API
1. Clonar proyecto
2. __yarn install__
3. Clonar el archivo __.env.template__ y renombrarlo a __.env__
4. Cambiar las variables de entorno

5. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar el proyecto: 
```
yarn star:dev
```

7. Ejecutrar SEDD
```
http://localhost:3000/api/seed
```
